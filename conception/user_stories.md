# User Stories - Site E-commerce de Café

## Utilisateur non connecté

| En tant que              | Je souhaite accéder à…    | Afin de…                                                  |
| ------------------------ | ------------------------- | --------------------------------------------------------- |
| Utilisateur non connecté | la page d'accueil         | découvrir l'offre de café et naviguer sur le site         |
| Utilisateur non connecté | le catalogue des produits | consulter les cafés disponibles et leurs caractéristiques |
| Utilisateur non connecté | le détail d'un produit    | voir la description, prix et avis avant achat             |
| Utilisateur non connecté | les filtres de recherche  | trouver un café par origine, grain ou torréfaction        |
| Utilisateur non connecté | les avis clients          | m'informer sur la qualité des produits                    |
| Utilisateur non connecté | le panier temporaire      | préparer ma commande avant de créer un compte             |
| Utilisateur non connecté | la page d'inscription     | créer un compte pour finaliser mes achats                 |
| Utilisateur non connecté | la page de connexion      | accéder à mon compte existant                             |

## Utilisateur connecté

| En tant qu'          | Je souhaite accéder à…             | Afin de…                                                  |
| -------------------- | ---------------------------------- | --------------------------------------------------------- |
| Utilisateur connecté | mon profil utilisateur             | modifier mes informations personnelles                    |
| Utilisateur connecté | la gestion de mes adresses         | ajouter/modifier mes adresses de facturation et livraison |
| Utilisateur connecté | mon historique de commandes        | consulter mes achats précédents                           |
| Utilisateur connecté | le détail d'une commande           | vérifier les informations d'une commande spécifique       |
| Utilisateur connecté | le suivi de mes commandes          | connaître le statut d'avancement de mes livraisons        |
| Utilisateur connecté | l'annulation de commande           | annuler une commande non encore validée                   |
| Utilisateur connecté | mes factures                       | télécharger mes justificatifs d'achat                     |
| Utilisateur connecté | mon panier persistant              | retrouver mes articles lors de ma reconnexion             |
| Utilisateur connecté | mes listes de favoris              | sauvegarder et organiser mes produits préférés            |
| Utilisateur connecté | la rédaction d'avis                | partager mon expérience sur les produits achetés          |
| Utilisateur connecté | la gestion de mes avis             | modifier ou supprimer mes commentaires                    |
| Utilisateur connecté | la gestion de mes methode paiement | modifier/ajouter ou supprimer des méthodes de paiement    |
| Utilisateur connecté | les demandes de retour             | retourner un produit non satisfaisant                     |
| Utilisateur connecté | le suivi de mes retours            | connaître l'avancement de mes demandes de remboursement   |
| Utilisateur connecté | mes préférences newsletter         | gérer ma souscription aux communications                  |
| Utilisateur connecté | la modification de mot de passe    | sécuriser mon compte                                      |
| Utilisateur connecté | la fonction de déconnexion         | sécuriser mon compte sur un appareil public               |

## Responsable produit

| En tant que         | Je souhaite accéder à…          | Afin de…                                                 |
| ------------------- | ------------------------------- | -------------------------------------------------------- |
| Responsable produit | la gestion des produits         | ajouter, modifier ou désactiver des cafés du catalogue   |
| Responsable produit | la gestion des stocks           | mettre à jour les quantités disponibles                  |
| Responsable produit | l'upload d'images               | ajouter des visuels attractifs aux produits              |
| Responsable produit | la gestion des catégories       | organiser et hiérarchiser le catalogue                   |
| Responsable produit | l'assignation de catégories     | classer les produits pour faciliter la navigation        |
| Responsable produit | la création de promotions       | mettre en place des offres spéciales                     |
| Responsable produit | la configuration des promotions | définir les conditions et périodes d'application         |
| Responsable produit | les statistiques de promotions  | analyser l'efficacité des offres                         |
| Responsable produit | la modération des avis          | valider et gérer les commentaires clients                |
| Responsable produit | les réponses aux avis           | répondre aux commentaires pour améliorer la satisfaction |
| Responsable produit | les rapports produits           | analyser les performances du catalogue                   |

## Administrateur

| En tant qu'    | Je souhaite accéder à…                  | Afin de…                                           |
| -------------- | --------------------------------------- | -------------------------------------------------- |
| Administrateur | la liste des utilisateurs               | consulter et gérer la base clients                 |
| Administrateur | la gestion des statuts utilisateur      | activer, suspendre ou supprimer des comptes        |
| Administrateur | l'attribution des rôles                 | gérer les permissions d'accès au système           |
| Administrateur | la suppression de comptes               | traiter les demandes de suppression RGPD           |
| Administrateur | toutes les commandes                    | superviser l'activité commerciale globale          |
| Administrateur | la gestion des statuts commande         | modifier l'état d'avancement des commandes         |
| Administrateur | l'annulation de commandes               | traiter les cas exceptionnels                      |
| Administrateur | la validation des commandes             | contrôler les ventes avant traitement              |
| Administrateur | la gestion des livraisons               | créer et assigner les expéditions                  |
| Administrateur | le suivi logistique global              | superviser toutes les livraisons en cours          |
| Administrateur | la saisie des numéros de suivi          | informer les clients de l'expédition               |
| Administrateur | l'historique des paiements              | consulter les transactions financières             |
| Administrateur | les paiements en échec                  | résoudre les problèmes de transaction              |
| Administrateur | la configuration des moyens de paiement | adapter les options de paiement disponibles        |
| Administrateur | les demandes de retour                  | traiter et valider les retours produits            |
| Administrateur | la gestion des remboursements           | effectuer les remboursements validés               |
| Administrateur | les statistiques de retours             | analyser la qualité et satisfaction client         |
| Administrateur | le tableau de bord global               | suivre les KPI et performances du site             |
| Administrateur | la configuration système                | maintenir et paramétrer la plateforme              |
| Administrateur | l'export de données                     | générer des rapports et analyses                   |
| Administrateur | les logs système                        | diagnostiquer et résoudre les problèmes techniques |
